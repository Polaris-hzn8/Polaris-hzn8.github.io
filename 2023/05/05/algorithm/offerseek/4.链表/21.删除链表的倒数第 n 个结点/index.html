删除链表的倒数第 n 个结点

  • 原题链接：https://leetcode.cn/problems/SLwz0R/

给定一个链表，删除链表的倒数第 n 个结点，并且返回链表的头结点。

1.计算链表长度

一种最容易想到的方法：（无虚拟头结点）

  1. 首先从头节点开始对链表进行一次遍历，得到链表的长度length
  2. 然后从头节点开始对链表进行一次遍历，当遍历到第L−n+1个节点时，其下一个就是需要删除的节点
  • 时间复杂度：O(L)其中L为链表的长度
  • 空间复杂度：O(1)

2.利用栈

主要思路：（无虚拟头结点）

  1. 在遍历链表的同时将所有节点依次入栈
  2. 根据栈「先进后出」的原则，弹出栈的第n个节点就是需要删除的节点，并且目前栈顶的节点就是待删除节点的前驱节点
  3. 这样删除操作就变得十分方便了
  • 时间复杂度：O(L) L为栈的长度
  • 空间复杂度：O(L) L为栈的长度，主要是栈的开销

3.双指针

主要思路：双指针使用常数空间解决

  1. 由于需要找到倒数第n个结点 可以使用两个指针 first 与 second 同时对链表进行遍历
  2. 并且 first 比 second 超前n个节点
  3. 当 first 遍历到链表的末尾时，second 就恰好处于倒数第 n 个节点
  • 时间复杂度：O(L)
  • 空间复杂度：O(1)
